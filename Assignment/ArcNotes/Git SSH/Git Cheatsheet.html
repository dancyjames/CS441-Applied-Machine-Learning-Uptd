Git Cheatsheet

Git status

```

git status

```

Get files from an existing online folder into local

```

git fetch

```

Create new branch, copy from main branch

  1. Create branch in git online (Goto Branches > New Branch under main)
  1. in CLI, “git pull” (to pull latest branches from online)

3. Point to new branch “git checkout Preethi-v3”

4.  Move data from main to new branch “git merge main”

45. Within this branch “source activate cs411” to activate environment.
